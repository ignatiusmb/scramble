## Getting Started

### Install with npm

```bash
npm install @ignatiusmb/scramble
```

### Or get it through a CDN

```html
<script src="https://cdn.jsdelivr.net/gh/ignatiusmb/scramble.js@1.0.0/lib/scramble.min.js"></script>
```
